Question-1: What are the different ways to select an element in the DOM?

Ways to select an element in the DOM: You can grab elements using getElementById(), getElementsByClassName(), getElementsByTagName(), or the more flexible querySelector() and querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent: innerHTML gives you the whole content, including HTML tags, innerText only shows visible text, and textContent gets all text, even if it's hidden, without processing HTML.

Question-4: What is event bubbling in the DOM?

When an event happens on an element, it first runs on that element and then moves up to its parents unless stopped.

Question-5: How do you create, add, and remove elements using JavaScript?

We create elements with document.createElement(), add them with appendChild() or insertBefore(), and remove them using removeChild() or remove().
